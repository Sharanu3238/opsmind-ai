\# OpsMind AI - Enterprise SOP Agent



This project is a Retrieval-Augmented Generation (RAG) system that allows users to upload SOP documents and query them intelligently.



\## Features

\- PDF upload and parsing

\- Text chunking

\- Embedding storage in MongoDB

\- Semantic retrieval

\- Context-based answering



\## Tech Stack

\- FastAPI

\- MongoDB Atlas

\- Gemini API (planned)
